Rules and Regulations:

Voting


  • Each student can cast only one vote.
  • Once voting starts, unions can not be created.
  • Students can vote by logging in with their usernames.
  • once the vote is casted, it can not be undone.
  • The voter can only choose one Union.
  • The voting terminal can not be accessed before the adimin sets it.

Creating Unions:


  • Each Union must have 5 members.
  • A Union can have only one Head.
  • The five members work together under the Head.
  • Unions can be created till the date alloted.
  • Only the Head must create the Union.
  • If someone else creates the Union, he/she will be flagged.
  • If there are any complications while creating the unions, the faculties concerning the voting process are to be contacted.
  • Union ID must be Unique for every Union.
  • Union Name must be unique for every Union.
